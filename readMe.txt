Este reposítório contém a impementação de um timeliner, que é constituído por linhas e eventos

Passo 1 -
Criar dados aleatórios de um timeline:
  python3 gerador.py <quantidade_eventos> <tamanho_maximo_evento> <ano_inicio> <ano_final> "<nome>"

Passo 2 -
Criar o timeline utilizando o .txt gerado no passo dois:
  python3 timelinemaker.py <entrada.txt> 


Para realizar o primeiro teste:
  cd First_test
